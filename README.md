## 📌 Proyecto de Infraestructura Empresarial

Proyecto desarrollado durante las prácticas del ciclo formativo de **Administración de Sistemas Informáticos en Red (ASIR)**, orientado al diseño, despliegue y administración de una infraestructura tecnológica completa que simula un entorno empresarial real.

La solución integra virtualización, sistemas Windows y Linux, bases de datos empresariales, contenedores Docker, monitorización avanzada, automatización, servicios web y herramientas de ciberseguridad dentro de una arquitectura centralizada, escalable y segura.

---

## 🧠 Resumen del Proyecto

### 🏗️ Infraestructura
- Proxmox VE como plataforma principal de virtualización  
- Máquinas virtuales Ubuntu Server y Kali Linux  
- Windows Server para dominio y administración centralizada  
- Raspberry Pi para servicios auxiliares y monitorización  

---

### ⚙️ Servicios implementados
- Active Directory y políticas de grupo (GPOs)  
- Servidor FTP  
- Servidor de correo interno  
- Oracle Database 21c Multitenant  
- PostgreSQL  
- Docker + Portainer  
- Proxy inverso con NGINX  
- Despliegue de +10 sitios web  
- Landing page centralizada de servicios  

---

### 📊 Monitorización y seguridad
- Grafana  
- Prometheus  
- Node Exporter  
- cAdvisor  
- Netdata  
- Wireshark  
- Nmap  
- Laboratorio de análisis de red con Kali Linux  

---

### 🤖 Automatización
- Triggers y procedimientos almacenados  
- Auditoría automática de sistemas  
- Backups programados  
- Control de actividad y logs  
- Integración Excel ↔ PostgreSQL  
- Formularios VBA para gestión académica

## 🧩 Aplicación desarrollada

### 📊 Sistema de gestión académica (Excel VBA)

Aplicación desarrollada en Microsoft Excel VBA para la gestión administrativa y académica dentro del entorno simulado.

**Funciones principales:**
- Autenticación de usuarios  
- CRUD completo (alta, baja, modificación)  
- Búsqueda avanzada de registros  
- Auditoría de operaciones  
- Integración con PostgreSQL  
- Copias de seguridad automatizadas
